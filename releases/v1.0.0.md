# Pi Desktop — First Release 🎉

A native desktop shell for the **Pi Coding Agent** CLI, with multi-session chat,
streaming, tools, a docked terminal, and an extension-first architecture.

> **Note:** This is a personal fork of the excellent
> [`gustavonline/pi-desktop`](https://github.com/gustavonline/pi-desktop). Full credit
> to the original authors — this fork carries my own improvements, bug fixes, and a
> Windows-focused build/release workflow.

This first release packages up everything I've layered on top of the upstream project.

---

## ✨ What's new in this fork

### 1. Auto session naming (via LLM)
New chat sessions now get **descriptive titles generated automatically** instead of
staying as "New session". The app reads your model config, calls your configured
OpenAI-compatible provider, and names the session from your first message — with a
graceful word-based fallback if anything fails.

- Matches the **active provider + model** against your `models.json` to use the right credentials
- Re-checks before applying, so it never clobbers a name you set manually or renames the wrong tab
- Sanitized output, debug-traceable, and resilient to network/provider errors

### 2. Smoother streaming in long sessions
Fixed a stutter that appeared after the app ran for a while. Every streaming token
used to trigger a full chat re-render; renders are now **coalesced to one per animation
frame** (~60fps). Net result: streaming stays smooth even as sessions grow long.

### 3. Windows experience improvements
- **Windows-style window controls** on the sidebar (minimize / maximize / close)
- Sidebar layout tuned for Windows — controls on the right, toggle on the left
- **Clear chat** button in the composer toolbar
- Fixed `pi.cmd` discovery on Windows for environments where PATH lookup is unreliable

### 4. UI polish
- Provider picker styling, accent color, and the Inter font family
- Micro-interactions throughout
- Wider surface contrast, larger minimum font size, and stronger border visibility
- Added OpenAI-compatible provider configuration in Settings
- Streamlined the packages page (removed the "recommended" clutter)

## 📦 Download

This fork builds **Windows only**. Grab the installer from the assets below:

- **`Pi Desktop_1.0.0_x64-setup.exe`** — NSIS installer for Windows x64

> For macOS or Linux, build from source (see the README) or use the upstream releases.

### ⚠️ Unsigned build (Windows SmartScreen)
Builds are not code-signed, so Windows may show a SmartScreen warning on first launch:
1. Click **More info**
2. Click **Run anyway**

---

## ⚙️ Requirements

The app needs the **Pi Coding Agent CLI** on your machine. If it's missing, the app
shows an onboarding card on first launch. Install it with:

```bash
npm install -g @earendil-works/pi-coding-agent
```

(Node.js ≥ 22 recommended.)

---

## 🚀 Getting started

1. Download and run the `.exe` installer above
2. Open **Pi Desktop**
3. If prompted, install the `pi` CLI (command above) and click **Retry**
4. Pick a project and start chatting — sessions will auto-name themselves
