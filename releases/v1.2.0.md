## ✨ What's new

### 1. Real PTY terminal backend
The docked terminal is now backed by a **native PTY** (`portable-pty` Rust bridge) instead of xterm.js addons. Proper resize, input handling, and isolated shell execution — no more terminal commands leaking into the chat timeline.

### 2. Auto session naming — works with every model
Sessions get descriptive LLM-generated titles automatically. Uses `pi --print` to generate the title, which means pi handles credentials, proxy routing, TLS, and API format. Works with **any model in the picker**.

### 3. Smoother streaming
Streaming text renders are coalesced to one per animation frame (~60fps) via `requestAnimationFrame`. Fixes progressive stutter in long sessions.

### 4. Combined Windows + macOS build
Single `workflow_dispatch` trigger now builds both Windows (`.exe`) and macOS Intel (`.dmg`) in parallel, published to a single draft release.

### 5. Windows polish
- JetBrains Mono as default app font
- Windows-style window controls on sidebar

## 📦 Download

### Windows
- **`Pi Desktop_*.exe`** — NSIS installer

### macOS
- **`Pi Desktop_*.dmg`** — unsigned Intel build (cross-compiled on Apple Silicon)

### ⚠️ Unsigned builds
**Windows:** Click **More info** → **Run anyway** on SmartScreen.

**macOS:** Remove quarantine after mounting:
```bash
xattr -cr "/Applications/Pi Desktop.app"
```
