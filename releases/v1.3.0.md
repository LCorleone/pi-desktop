# Pi Desktop v1.3.0 — UI Polish & Thinking Overhaul ✨

A major visual refresh: overhauled thinking/tool display, 24+ CSS fixes,
auto session naming fix, and grey app icon.

> **Note:** This is a personal fork of [`gustavonline/pi-desktop`](https://github.com/gustavonline/pi-desktop).

---

## ✨ What's new

### Thinking — completely overhauled

Thinking blocks are now polished end-to-end:

- **Shimmer animation** — smooth text-gradient effect replaces the old per-character sweep
- **Auto-expand** — reasoning content opens automatically during streaming, collapses when the answer arrives
- **Collapsed state** — shows a subtle `▸ Thought` caret indicator (hover to brighten), no more bold "Thinking…"
- **Content always visible** — thinking defaults to expanded when loaded from history
- **Tool status dots** — colored indicators (green/red/blue) on tool rows instead of text labels

### Tools — cleaner & more informative

| Before | After |
|--------|-------|
| `Ran ls -la /usr/local/bin` | 🖥️ **bash** `ls -la /usr/local/bin` |
| `Read /very/long/path/to/file.ts` | 📄 **read** `file.ts` |
| `Write /path/config.json` | 📝 **write** `config.json` |

- **Tool-type icons** — 6 categories (terminal, read, write, edit, search, default)
- **Clean labels** — tiny pill tags (bash, read, write, etc.) between icon and content
- **Tooltips** — hover any collapsed tool row to see the first 300 chars of output
- **Filename only** — read/write/edit tools show just the filename, not the full path

### Auto session naming — fixed

When asking a question, the session name was becoming the AI's **answer** instead of a title. Fixed the prompt to explicitly say "Do NOT answer — just name it."

### 24 CSS polish fixes

- **7px/9px illegible text** bumped to readable sizes (composer labels, attachment thumbnails)
- **Focus rings** added to all interactive buttons (composer, send, stop, message actions)
- **Working indicator** — "Pi is working..." now 12px bolder text, faster animation
- **Model picker** — unauthenticated providers no longer invisible, better auth hint contrast
- **Sidebar** — workspace pills visible again (raised opacity & saturation)
- **Status labels** — tool status bumped to 12px bold

### App icon — grey gradient

New neutral grey gradient icon replacing the vibrant blue. Blends with any desktop theme.

### Terminal stability

- Fixed bogus "shell exited" on workspace switch (generation tracking)
- PTY temp file cleanup on all error paths
- 30s timeout on `pi --print` title generation
- PTY sessions killed on app quit

## 📦 Download

This fork builds **Windows** (`.exe`) and **macOS** (`.dmg`). Both triggered from a single workflow.

### ⚠️ Unsigned builds

**Windows:** Click **More info** → **Run anyway** on SmartScreen.

**macOS:** `xattr -cr "/Applications/Pi Desktop.app"` after mounting.
